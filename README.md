# oneMoreSerializer  

Задача из универа. Нужно было написать расширяемый бинарный сериализатор для пересылки данных по сети и возможности проверить корректность данных на принимающей стороне.
В частности среди требований были: сжатие данных и возможность кастомизации сериализации конкретного класса. Код для сжатия был взят со стороннего ресурса, всё остальное моё.

Помимо бинарного сериализатора, в проекте есть текстовый сериализатор, разработанный в рамках спецкурса "Школа промышленной разработки".
